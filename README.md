🚀 Sh4rk Optimizer SuiteO

Sh4rk Optimizer Suite é uma coleção de scripts e ajustes (tweaks) de baixo nível 
desenvolvidos para extrair a performance máxima do hardware. Focado especialmente em
reduzir o input lag, remover telemetria desnecessária e estabilizar o FPS em jogos como 
Roblox, Free Fire e PES.🛠️ Funcionalidades PrincipaisRegistry Tweaks: Ajustes finos no 
registro do Windows para prioridade de CPU e GPU.Debloat: Remoção de serviços nativos 
inúteis que consomem RAM em segundo plano.Network Optimization: Scripts para reduzir a 
latência e estabilizar o ping.Android Power: Comandos via ADB para otimização de bateria e
performance em dispositivos móveis.📁 Estrutura do RepositórioPastaDescrição/windowsScripts .bat, .ps1 e arquivos .reg para Windows 10/11./androidComandos ADB e guias de otimização para mobile./toolsPequenos utilitários de terceiros validados para performance.🚀 Como Usar[!CAUTION]Aviso: Sempre crie um Ponto de Restauração antes de aplicar qualquer tweak de registro.
